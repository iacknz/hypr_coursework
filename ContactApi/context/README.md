# Context

This folder is the **project knowledge base** — it explains everything about the Contact API in plain language. Think of it as the "about this project" section that helps anyone (including AI assistants) understand what's going on.

## Files

| File | What It Explains |
|---|---|
| `About.md` | What the project is, what problem it solves, who it's for, and what limits it has |
| `Architecture.md` | How the project is built, how the pieces fit together, and how data flows through the system |
| `Implementation.md` | What programming languages and tools are used, what versions, and what external packages are needed |

## Who Should Read These

- **New team members** — Get up to speed quickly on what this project is all about
- **AI assistants** — Read these files to understand the project before helping with code
- **Anyone curious** — Get a clear picture of the project without needing to read the code itself
